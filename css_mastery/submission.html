Challenge 1: Animating the Ansah Animation

⏰ Due: 5:30 PM (End of Class today)

📋 Mission

Modify an HTML file's embedded stylesheet to create a smooth fraction animation that moves across the screen with color transitions!

🎯 Requirements

  • Create a keyframe called "moveFraction" that animates the fraction through three stages:
  • At 0%: Set font color to yellow and translate to position (-355px, 60px) - fraction starts off the left side of screen
  • At 50%: Translate to (0px, 60px) - fraction appears just below and right of the question
  • At 100%: Set font color to red and translate to (0px, 0px) - fraction aligns next to the = sign
  • Animation Timing: Start the moveFraction animation 1 second after CSS loads in the #answer rule

Challenge 2: I ♥ NY T-Shirt Z-Index

⏰ Due: 5:30 PM (End of Class today)

📋 Mission

Recreate the iconic "I ♥ NY" logo on a t-shirt using CSS positioning and z-index properties!

🎯 Requirements

  • Relative Positioning: In the img CSS rule, use relative positioning to place the t-shirt image 10 pixels to the right of its default location
  • Absolute Positioning: In the .first, .heart, and .last CSS rules, use absolute positioning to place "I", "♥", and "NY" in the correct configuration on top of the t-shirt
  • Z-Index Layering: Ensure text elements appear above the t-shirt using proper z-index values

💻 HTML Structure

<span class="first words">I</span> <span class="heart">♥</span> <span class="last words">NY</span> <div> <img src="[white tee shirt image]" alt="T-Shirt"> </div>

🌟 CSS Pro Tips & Resources

🎨 Animation Best Practices

Use transform over position properties - Animations using transform are GPU-accelerated and perform much better than animating top, left, etc.

📐 Z-Index Guidelines

Only works on positioned elements - Remember that z-index only affects elements with a position value other than static (the default).

🔧 Debugging Tips

Use browser DevTools - Right-click and "Inspect Element" to see computed styles, test changes in real-time, and visualize positioning!
